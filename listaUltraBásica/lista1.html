a) O que é Internet e WEB? São as mesmas coisas? Explique de forma clara e evidencie as diferenças.

a Internet é a rede que conecta todos os dispositivos, enquanto a Web é um serviço que permite a visualização de informações nessa rede. Embora estejam interrelacionadas, elas não são a mesma coisa.

b) O que é arquitetura cliente-servidor utilizada na web ? Explique o carregamento de uma página passo a passo.

Esse modelo cliente-servidor é essencial para a funcionalidade da web, permitindo uma comunicação eficaz entre o usuário e os servidores que hospedam o conteúdo online.

c) O que é um tag HTML?

Uma tag HTML funciona como uma "instrução" que ajuda a estruturar e formatar o conteúdo de uma página da web. Cada tag informa ao navegador como exibir elementos, como textos, imagens, links e outros conteúdos. Dessa forma, as tags são essenciais para a criação de layouts e a apresentação de informações na web.

d) O que é HTML e qual a estrutura básica de um documento HTML?

HTML (HyperText Markup Language) é a linguagem utilizada para desenvolver páginas na web. Ela permite organizar e estruturar o conteúdo exibido no navegador, incluindo elementos como textos, imagens, links e muito mais. 

        <!DOCTYPE html>
        <html lang="pt-BR">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Título da Página</title>
        </head>
        <body>
            <h1>Título Principal</h1>
            <p>Este é um parágrafo de exemplo.</p>
        </body>
        </html>

f) Explique como é criado um link no html sobre um parágrafo.

Para criar um link em HTML dentro de um parágrafo, usamos a tag "". Nela, definimos o atributo href com o endereço para onde o link vai levar.
